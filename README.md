# MathCut (매쓰컷)

> 수학 교재용 문제 이미지 생성 서비스 — 모눈종이 위에 도형을 정밀하게 그리고, AI로 이미지를 완성하세요.

---

## 서비스 소개

**MathCut**은 수학 교재 제작자, 교사, 출판사를 위한 **수학 문제 이미지 편집 도구**입니다.

기존에는 수학 도형 이미지를 만들기 위해 복잡한 그래픽 도구나 LaTeX을 직접 다뤄야 했습니다. MathCut은 모눈종이 위에서 **클릭만으로 도형을 그리고**, 길이·각도 같은 **수학적 가이드를 자동으로 표시**하며, 완성된 이미지를 **PNG·SVG·PDF로 바로 내보낼 수 있는** 올인원 편집 환경을 제공합니다.

또한 손으로 스케치한 그림이나 업로드한 이미지를 **AI가 분석해 도형으로 자동 변환**하는 기능도 갖추고 있어, 빠른 초안 작성이 가능합니다.

---

## 주요 기능

### 도형 그리기

모눈종이 격자에 **스냅(Snap)** 되어 정확한 위치에 도형을 배치할 수 있습니다. 클릭으로 꼭지점을 찍으면 도형이 자동으로 완성됩니다.

#### 지원 도형 (30가지 이상)

| 분류 | 도형 |
|------|------|
| **삼각형** | 자유 삼각형, 정삼각형, 직각삼각형, 이등변삼각형 |
| **사각형** | 자유 사각형, 정사각형, 직사각형, 마름모, 평행사변형, 사다리꼴 |
| **원** | 원 (반지름 / 지름 측정 모드 선택 가능) |
| **직선류** | 선분, 반직선, 직선, 각도선 |
| **화살표** | 직선 화살표, 곡선 화살표 |
| **다각형** | 정다각형 (변 개수 3~12개 자유 설정), 자유도형 |
| **점** | 점, 점-위의-점 (도형 위의 특정 점 표시) |

- **삼각형·사각형 특수 유형**은 두 점만 찍으면 나머지 꼭지점을 자동 계산합니다.
  (예: 정삼각형은 밑변 두 점 → 꼭지점 자동 생성)
- **정다각형**은 변 개수를 3~12개로 설정할 수 있습니다.
- **점-위의-점**은 다른 도형의 변 위에 붙어 함께 이동합니다.

---

### 가이드 (수학적 표기)

도형 위에 수학 교재에서 흔히 보는 표기를 추가할 수 있습니다.

| 가이드 종류 | 설명 |
|------------|------|
| **길이** | 두 점 사이의 거리를 선과 숫자로 표시. 단위(cm) 포함 가능 |
| **각도** | 세 점이 이루는 각도를 호(arc)와 숫자로 표시. 단위(°) 포함 가능 |
| **텍스트** | 자유롭게 텍스트 라벨 추가. LaTeX 수식 입력 지원 |
| **빈 칸** | 학생이 답을 적는 공란 표시. cm / ° 단위 선택 가능 |

**자동 가이드 표시 옵션:**

- **길이 자동 표시**: 도형의 각 변 길이를 한 번에 표시
- **각도 자동 표시**: 직각만 표시 / 모든 각도 표시 중 선택
- **높이 자동 표시**: 삼각형·사다리꼴·평행사변형에 높이 가이드 자동 생성
- **꼭지점 이름 표시**: A, B, C … 또는 ㄱ, ㄴ, ㄷ … 자동 라벨링
- **직각 마커**: 90° 꼭지점에 정사각형 기호 자동 표시

---

### 모눈종이 표시 모드

| 모드 | 설명 |
|------|------|
| **격자 (grid)** | 격자선 + 소격자 보조점 표시 (기본값) |
| **점 (dots)** | 격자 교차점만 점으로 표시 |
| **없음 (none)** | 배경 없이 백지 상태 |

격자 선 색상과 배경색도 커스텀으로 변경할 수 있습니다.

---

### 컷(Cut) 관리

하나의 작업 공간 안에서 여러 개의 **컷(Cut)** 을 만들어 관리할 수 있습니다.

- 컷 추가 / 복제 / 삭제
- 컷 간 빠른 전환
- 좌측 썸네일로 전체 컷 한눈에 확인

---

### AI 스케치 분석

손으로 스케치한 그림이나 이미지를 업로드하면 AI가 분석해 도형과 가이드를 자동으로 생성합니다.

1. 모달에서 스케치 이미지 업로드 또는 직접 그리기
2. 도형 유형 힌트 입력 (선택사항)
3. AI 분석 → 도형 + 가이드 자동 생성
4. 결과를 캔버스에 바로 적용

---

### 내보내기

완성된 이미지를 다양한 포맷으로 내보낼 수 있습니다.

| 포맷 | 특징 |
|------|------|
| **PNG** | 투명 배경 지원, 커스텀 해상도 설정 |
| **SVG** | 벡터 그래픽, LaTeX 수식 및 폰트 임베딩 포함, 무손실 확대 |
| **PDF** | 용지 크기 자동 조정, 인쇄 최적화 |

- 해상도(가로 × 세로)를 직접 입력해 출력 크기를 조정할 수 있습니다.
- 흑백 변환(Grayscale) 옵션으로 흑백 인쇄용 이미지를 생성할 수 있습니다.
- 격자 배경 포함/제외를 선택할 수 있습니다.

### 편집 기능

| 기능 | 설명 |
|------|------|
| **선택 / 이동** | 도형을 클릭해 선택하고 드래그로 위치 이동 |
| **꼭지점 수정** | 선택된 도형의 꼭지점을 드래그해 모양 변경 |
| **실행 취소 / 다시 실행** | Ctrl+Z / Ctrl+Y (히스토리 최대 100단계) |
| **컨텍스트 메뉴** | 우클릭으로 복사, 삭제 등 빠른 조작 |
| **레이어 순서 변경** | 도형의 앞/뒤 순서 조정 |
| **인라인 텍스트 편집** | 텍스트 가이드를 더블클릭해 직접 편집 |
| **LaTeX 수식 입력** | 꼭지점 이름, 텍스트 가이드에 LaTeX 수식 입력 가능 |
| **줌** | 캔버스 배율 100~200% 조정 |
| **스냅** | 일반 스냅(20px 격자) / 정밀 스냅(5px 소격자) |

## 시작하기

### 요구 사항

- Node.js 18+
- npm 9+

### 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행 (http://localhost:5174)
npm run dev

# 프로덕션 빌드
npm run build

# 빌드 결과물 미리보기
npm run preview
```

### 백엔드 실행 (선택사항)

AI 스케치 분석, 저장/로드 기능을 사용하려면 `mathcut-api`를 실행해야 합니다.

- 저장소: [https://github.com/Geeehyun/mathcut-api](https://github.com/Geeehyun/mathcut-api)

```bash
# mathcut-api 디렉토리에서
OPENAI_API_KEY=sk-... ./gradlew bootRun
# Base URL: http://localhost:9090
```
